     9|## Overview
    10|
    11|A personal finance dashboard for tracking income, expenses, and categorized transactions. Built with **React**, **TypeScript**, and **Styled Components** for a polished, responsive interface.
    12|
    13|## Features
    14|
    15|- Transaction management (income/expense)
    16|- Category-based filtering and search
    17|- Summary dashboard with balance overview
    18|- Modal-based transaction creation
    19|- Responsive design
    20|
    21|## Tech Stack
    22|
    23|| Technology | Purpose |
    24||---|---|
    25|| **React** | UI library |
    26|| **TypeScript** | Type safety |
    27|| **Styled Components** | CSS-in-JS styling |
    28|| **Vite** | Build tool |
    29|| **Zod** | Form validation |
